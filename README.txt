Scikit-learn Tutorial (3/6/2017)


This repository consist a few codes and documents for the Scikit-learn library tutorial regarding with Machine Learning modeling.

The purpose of this repository is to assist users to get engaged with machine learning algorithms by utilizing one of the most famous ML library: Scikit-learn.

By following instructions, users will be able to gauge a sense of how to build ML projects using algorithms and its methods for predictions, clustering, and evaluations.
